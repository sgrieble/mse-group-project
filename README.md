# MSE Group Project

## 0. Setup Project
Create a virtual environment. After activating the virtual environment, install the dependencies:
```
pip install -r requirements.txt
```
To get the user interface of the search engine running, navigate to the folder `app` and execute `main.py`:
```
python3 main.py
```
